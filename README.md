## 🛠 Installation | نصب

1. Clone the repository:
   ```bash
   git clone [repository-url]
   cd falcon-client
   ```

2. Install dependencies:
   ```bash
   npm install
   ```

3. Start the development server:
   ```bash
   npm run dev
   ```

4. Open [http://localhost:5173](http://localhost:5173) to view it in your browser.
